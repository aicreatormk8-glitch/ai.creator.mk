# Marina Kalashnikova — AI Creator Studio

Преміальний односторінковий сайт-портфоліо у **кінематографічному AI-стилі** для
**Marina Kalashnikova** (`@aicreatormk8`) — AI Creator та спеціалістки з цифрової автоматизації.

Dark mode · glassmorphism · neon glow · fullscreen cinematic hero (`hero.jpg`) ·
анімоване боке · плавні scroll-анімації.

## Структура

```
index.html   # розмітка + вбудований SVG-спрайт іконок
styles.css   # дизайн-система на CSS-змінних (cinematic dark + glass)
script.js    # навігація, боке-частинки, scrollspy, фільтр портфоліо, лічильники, форма
hero.jpg     # fullscreen-фон героя
```

## Секції

- **Hero** — fullscreen `hero.jpg`, cinematic-скрими, анімоване боке, статистика, CTA
- **Marquee** — рядок напрямів послуг, що рухається
- **Послуги** — 3 напрями (AI-візуал та контент · Сайти · Автоматизація) з повним переліком
- **Портфоліо** — сітка з фільтром за категоріями (AI-фото / відео / моделі / сайти / автоматизація / бренд)
- **Процес** — 4 етапи від брифу до деплою
- **Тарифи** — пакети Старт / Studio / Premium
- **Відгуки** — соціальний доказ
- **Контакти** — форма (mailto) + швидкі CTA + прямі контакти
- **Footer** — бренд, послуги, студія, соцмережі

## Доступність

WCAG-орієнтований: контраст, `:focus-visible`, SVG-іконки (без emoji), aria-мітки,
`prefers-reduced-motion`, семантична структура, тач-цілі ≥44px.
